# Vehicle Service Management System
A Java-based application for managing vehicle services using JDBC and MySQL.

## Features
- Customer Management
- Vehicle Management
- Service Management
- Database operations using JDBC

## Project Structure
- **Model**: Customer, Vehicle, Service classes
- **DAO**: Data Access Objects for database operations
- **DB**: Database connection handling
- **Main**: VehicleService.java - Main application class

## Technologies Used
- Java
- JDBC
- MySQL

## Database Setup
1. Create MySQL database
2. Update database connection details in DB configuration
